# 组装指南

本文档介绍 InkSight 的硬件连接与上电检查流程，目标是让你在 10-20 分钟内完成可用硬件组装。

## 1. 组装前准备

- 主控板：ESP32-C3 SuperMini（推荐）
- 屏幕：4.2 英寸 SPI E-Paper（400x300）
- 连接线：杜邦线 6 根以上
- 供电：USB-C（调试期建议）或 LiFePO4 + TP5000（量产/长期使用）

## 2. 连接顺序建议

建议按以下顺序连接，便于定位故障：

1. 先接电源（VCC/GND）
2. 再接 SPI 主信号（CLK、DIN、CS）
3. 最后接控制线（DC、RST、BUSY）

这样即使出现黑屏或刷新异常，也能更快判断是供电问题还是信号线问题。

## 3. 首次上电检查

首次上电建议仅使用 USB 供电，不要同时接电池。检查项：

- ESP32-C3 能被电脑识别串口
- 屏幕上电后有初始化闪屏（或至少有刷新动作）
- 串口日志无持续报错（如初始化失败、内存错误）

## 4. 常见硬件问题

### 屏幕无显示

- 检查 GND 是否共地
- 检查 RST/BUSY 是否接反
- 检查屏幕型号是否为 SPI 版本

### 刷新异常（花屏/局部不刷）

- 优先检查 CLK/DIN/CS 稳定性
- 缩短连接线长度
- 排查供电电压是否过低

### 启动重启循环

- 常见于供电不足或电池模块输出不稳
- 调试阶段请先改用 USB 直供
